## 📋 **خطوات الاستخدام الفعلية:**

### **1. التشغيل الأول:**
```bash
npm run dev
# ثم الذهاب إلى http://localhost:3010
```

### **2. تسجيل الدخول:**
- أنشئ حساب عبر `POST /api/auth/register`
- أو سجل دخول

### **3. إعداد البيانات الأساسية:**
1. أضف موردين من `/dashboard/suppliers`
2. أضف منتجات من `/dashboard/products`
3. أضف عملاء من `/dashboard/customers`

### **4. البدء في العمل:**
1. سجل مشتريات من `/dashboard/purchases`
2. سجل مبيعات من `/dashboard/sales`
3. سجل دفعات من `/dashboard/payments`
4. اعرض التقارير من `/dashboard/reports`
