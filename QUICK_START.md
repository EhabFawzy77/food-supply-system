# 🚀 دليل البدء السريع

## ⚡ التشغيل في 5 دقائق

### **1. التثبيت والتشغيل:**
```bash
# تثبيت المتعلقات
npm install

# تشغيل خادم التطوير
npm run dev

# سيعمل على http://localhost:3010
```

---

## 📝 متطلبات البيئة

### **ملف `.env` المطلوب:**
```env
# اسم قاعدة البيانات
MONGODB_URI=mongodb://localhost:27017/food_supply_system

# مفتاح التشفير
JWT_SECRET=your-secret-key-here

# البيئة
NODE_ENV=development

# المنفذ (اختياري)
PORT=3010
```

## 📊 إعداد البيانات

### **خطوة 1: أضف موردين**
1. اذهب إلى `/dashboard/suppliers`
2. اضغط "إضافة مورد"
3. أدخل البيانات:
   - الاسم
   - رقم الهاتف
   - البريد الإلكتروني
   - العنوان
   - الرقم الضريبي

### **خطوة 2: أضف منتجات**
1. اذهب إلى `/dashboard/products`
2. اضغط "إضافة منتج"
3. أدخل البيانات:
   - اسم المنتج
   - الفئة
   - وحدة القياس (كجم، لتر، إلخ)
   - سعر الشراء
   - سعر البيع
   - الحد الأدنى للمخزون
   - اختر المورد

### **خطوة 3: أضف عملاء**
1. اذهب إلى `/dashboard/customers`
2. اضغط "إضافة عميل"
3. أدخل البيانات:
   - الاسم
   - اسم المتجر
   - رقم الهاتف
   - العنوان
   - حد الائتمان
   - نوع العميل (بيع بالتجزئة / بيع بالجملة)

## 💰 البدء في العمل

### **تسجيل شراء (مشتريات):**
1. اذهب إلى `/dashboard/purchases`
2. اضغط "إضافة مشتريات"
3. أدخل البيانات:
   - اختر المورد
   - أضف المنتجات والكميات
   - أدخل سعر الشراء
   - اختر طريقة الدفع
4. اضغط "حفظ"
   - سيتم تحديث المخزون تلقائياً

### **تسجيل بيعة (مبيعات):**
1. اذهب إلى `/dashboard/sales`
2. اضغط "إضافة مبيعة"
3. أدخل البيانات:
   - اختر العميل
   - أضف المنتجات والكميات
   - تحقق من أسعار البيع
4. اختر طريقة الدفع
5. اضغط "حفظ"
   - سيتم خصم من المخزون تلقائياً
   - سيتم حساب الربح تلقائياً

### **تسجيل دفعة:**
1. اذهب إلى `/dashboard/payments`
2. اضغط "إضافة دفعة"
3. أدخل البيانات:
   - النوع (دفعة من عميل / دفعة لمورد)
   - المبلغ
   - طريقة الدفع
   - التفاصيل (رقم الشيك، اسم البنك، إلخ)
4. اضغط "حفظ"
   - سيتم تحديث الديون تلقائياً

## 🔧 العمليات الأساسية

### **تعديل بيانات:**
1. اذهب إلى الصفحة المطلوبة (منتجات، عملاء، إلخ)
2. اضغط على رمز "تعديل" (✏️)
3. عدّل البيانات
4. اضغط "حفظ"

### **حذف بيانات:**
1. اذهب إلى الصفحة المطلوبة
2. اضغط على رمز "حذف" (🗑️)
3. أكد الحذف
4. سيتم حذف البيانات

### **البحث:**
1. استخدم حقل البحث في أي صفحة
2. اكتب اسم المنتج / العميل / المورد
3. ستظهر النتائج تلقائياً
