## 🏢 نظرة عامة

**Food Supply System** هو تطبيق متكامل لإدارة سلسلة التوريد الغذائية، يسمح بإدارة:
- 👥 المستخدمين والأدوار
- 🛒 العملاء والموردين
- 📦 المنتجات والمخزون
- 💰 المبيعات والمشتريات والدفعات
- 📊 التقارير والإحصائيات

---

## 🏗️ البنية المعمارية

### **Stack التقنيات:**
```
Frontend:  Next.js 16 + React 19 + TailwindCSS
Backend:   Next.js API Routes + Node.js
Database:  MongoDB + Mongoose ODM
Auth:      JWT + bcryptjs
UI:        Lucide React Icons + Recharts
```

### **هيكل المشروع:**
```
📦 food-supply-system
├── 📁 app/
│   ├── api/              # REST API endpoints
│   ├── dashboard/        # صفحات لوحة التحكم
│   └── (auth)/          # صفحات المصادقة
├── 📁 lib/
│   ├── models/          # Mongoose schemas
│   ├── middleware/      # Custom middleware
│   ├── utils/           # Utilities & helpers
│   └── mongodb.js       # DB connection
├── 📁 contexts/         # React Context providers
├── 📁 components/       # UI components
└── 📁 scripts/          # Setup & test scripts
```

## 🚀 كيفية الاستخدام

### **تشغيل المشروع:**
```bash
# تثبيت المتعلقات
npm install

# تشغيل خادم التطوير
npm run dev

# سيعمل على http://localhost:3010
```

### **تهيئة قاعدة البيانات:**
```bash
# يجب أن يكون MongoDB قيد التشغيل
# ثم إنشاء البيانات يدوياً عبر واجهة المستخدم:
# - أضف مستخدمين
# - أضف موردين
# - أضف منتجات
# - أضف عملاء
```

### **المتغيرات البيئية (.env):**
```env
MONGODB_URI=mongodb://localhost:27017/food_supply_system
JWT_SECRET=your-secret-key-change-in-production
NODE_ENV=development
```
