# 🎯 دليل البدء السريع - 5 دقائق

## ⚡ اليك ما تحتاج معرفته

### ✅ ما تم إضافته
نظام **طباعة فواتير كامل** يتضمن:
- إنشاء فاتورة من مبيعة
- عرض وتحرير الفواتير
- طباعة احترافية RTL عربية
- بحث وفلترة
- إحصائيات شاملة

### 🚀 كيفية الاستخدام (3 خطوات)

**الخطوة 1️⃣ - افتح الفواتير**
```
اذهب إلى: /dashboard/invoices
أو: من القائمة اضغط "الفواتير"
```

**الخطوة 2️⃣ - أنشئ فاتورة**
```
1. اضغط "فاتورة جديدة"
2. اختر مبيعة من القائمة
3. اضغط "إنشاء الفاتورة"
```

**الخطوة 3️⃣ - اطبع الفاتورة**
```
1. اضغط زر الطباعة 🖨️
2. ستفتح نافذة جديدة
3. اضغط Ctrl+P لطباعة أو حفظ
```

## 🔑 المعلومات المهمة

### المسارات (URLs)
```
قائمة الفواتير:     /dashboard/invoices
إنشاء فاتورة:       /dashboard/invoices/create
تفاصيل الفاتورة:    /dashboard/invoices/[id]
```

### API Endpoints
```
GET    /api/invoices              # جميع الفواتير
POST   /api/invoices              # فاتورة جديدة
GET    /api/invoices/[id]         # تفاصيل واحدة
PUT    /api/invoices/[id]         # تحديث
DELETE /api/invoices/[id]         # حذف
POST   /api/invoices/[id]/print    # تسجيل الطباعة
GET    /api/invoices/stats         # إحصائيات
```

### حقول الفاتورة الرئيسية
```
invoiceNumber   - رقم فريد (تلقائي)
customerName    - اسم العميل
items           - قائمة المنتجات
total           - المجموع
paymentStatus   - حالة الدفع (paid/unpaid/partial/overdue)
printCount      - عدد الطباعات
```

## 🎨 الواجهة الجديدة

### قائمة الفواتير
- 4 بطاقات إحصائيات ملونة
- حقل بحث سريع
- قائمة منسدلة للفلترة
- جدول الفواتير
- أزرار الإجراءات السريعة (عرض، طباعة، حذف)
- تصفح الصفحات

### تفاصيل الفاتورة
- بيانات العميل
- جدول المنتجات
- ملخص الحسابات في الشريط الجانبي
- محرر الملاحظات المباشر
- أزرار الطباعة والحذف
- إحصائيات الطباعة

### إنشاء فاتورة
- اختيار المبيعة من قائمة
- عرض تفاصيل المبيعة اختياري
- خيارات الدفع والملاحظات
- التحقق التلقائي من البيانات

## 🧪 اختبر بنفسك

### الاختبار السريع
```bash
# 1. شغّل الخادم
npm run dev

# 2. افتح المتصفح
http://localhost:3010/dashboard/invoices

# 3. انشئ فاتورة واطبعها
```

### اختبر API باستخدام cURL
```bash
# جلب الفواتير
curl -H "Authorization: Bearer YOUR_TOKEN" \
  http://localhost:3010/api/invoices

# إنشاء فاتورة
curl -X POST -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"saleId":"...", "paymentMethod":"cash"}' \
  http://localhost:3010/api/invoices
```

## ❓ الأسئلة الشائعة

### س: كيف أنشئ فاتورة؟
ج: من قائمة الفواتير اضغط "فاتورة جديدة"، اختر مبيعة، واضغط "إنشاء"

### س: هل يمكن تعديل الفاتورة؟
ج: نعم، يمكنك تعديل الملاحظات من صفحة التفاصيل

### س: هل تدعم الطباعة المتقدمة؟
ج: نعم، الطباعة واحفظ كـ PDF من المتصفح نفسه

### س: هل البيانات آمنة؟
ج: نعم، كل العمليات محمية بـ JWT والصلاحيات

### س: ماذا لو نسيت تسجيل الطباعة؟
ج: لا مشكلة، يتم تسجيل الطباعة تلقائياً
